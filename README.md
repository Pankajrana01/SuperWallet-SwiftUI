
# 🚀 SuperWallet App (SwiftUI)

A modern **Digital Wallet iOS application** built using **SwiftUI** with a focus on scalability, clean architecture, and real-world fintech use cases.

---

## 📱 Features

- 💳 Wallet Dashboard (Balance overview)
- 💸 Send & Receive Money
- ➕ Add Money / Top-up
- 📜 Transaction History
- 🔍 Transaction Details
- 👤 Profile & Settings

---

## 🛠 Tech Stack

- Swift 5
- SwiftUI
- MVVM Architecture
- Combine / Async-Await
- REST API Integration
- CoreData / Local Storage

---

## 🏗 Architecture

This project follows a **Clean MVVM Architecture**:

- **View** → SwiftUI UI components  
- **ViewModel** → Business logic & state management  
- **Model** → Data structures  
- **Services** → API & data layer
